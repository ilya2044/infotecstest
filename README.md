# Тестовое для ИнфоТеКС

## Запуск

Собрать проект и запустить с параметрами по умолчанию:

```bash
make run
```

Параметры по умолчанию:
- Файл журнала: `default.log`
- Уровень логирования: `MIDDLE`

## С кастомными параметрами

```bash
make run LOGFILE='название_файла' LOGLEVEL='УРОВЕНЬ'
```

Доступные уровни: `LOW`, `MIDDLE`, `HIGH`

## Команды в приложении

После запуска введите:

- `сообщение` — логировать сообщение с уровнем по умолчанию
- `сообщение LEVEL` — логировать с указанным уровнем (LOW/MIDDLE/HIGH), сообщение с уровнем ниже дефолтного не пройдет
- `level LEVEL` — изменить уровень по умолчанию
- `exit` — выход

## Пример

```bash
make run LOGFILE=test.txt LOGLEVEL=LOW
```

## Прочие команды

```bash
make test   # Запустить тесты
make lib    # Собрать только библиотеку
make app    # Собрать только приложение
make clean  # Удалить артефакты сборки
```
